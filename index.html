As Is

너무 많은 전시에 비해 카테고라이징, 페이징 기능이 없어 원하는 전시를 찾고 관람하기 어려운데다, 서비스의 이미지가 잘 나타나지 않는 점이 아쉬웠습니다.
기존 사이트

To Be

사용자의 편의를 위해 전시컨텐츠가 한눈에 들어오도록 큐레이팅하였고, 카테고리를 나눠 탭으로 확인 가능하도록 수정했습니다.
서비스 소개페이지에서는 이미지가 잘 나타나도록 브로슈어를 바탕으로 수정하였습니다.

Typo & Color

다양한 굵기와 정갈한 너비를 갖춘 프리텐다드를 사용하였고 로고와 키컬러는 기존 폰트 로고와 메인컬러를 베리에이션하여 사용했습니다.

UX Flow

화면설계서와 유저타입별 플로우차트를 작성하여 작업명세를 확실히 하고, 프로세스의 오류는 없는지 검증하는 과정을 거쳤습니다.

Design System

기본에 집중,
간결하지만 친숙하게!

Admin Dashboard

마치 유튜브처럼, 직접 전시 컨텐츠를 업로드하고 통계를 확인할 수 있는 어드민 대시보드 페이지를 함께 작업하였습니다. 심플하게 통계,편집,전시,계정 4가지 섹션으로 분류하여 사용자의 편의를 고려했습니다.

About Page

효과적인 서비스 이미지 각인을 위해서, 상단 hero부분은 영상을 활용하여 힘을 줬습니다. 하단으로는 B2B브로슈어의 내용을 기반으로 페이드업 효과로 고화질 이미지소스가 스크롤을 내릴 때 마다 발화되게끔 하여 보는이로하여금 서비스 소개가 지루하게 느껴지지 않도록 신경썼습니다.
